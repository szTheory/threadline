## Threat model

| ID | Threat | Severity | Mitigation in this plan |
|----|--------|----------|-------------------------|
| T-02-01-01 | Host passes attacker-controlled JSON into `set_config` ŌåÆ invalid JSON / oversized payload | medium | Document JSON must come from validated `ActorRef.to_map/1`; DB casts to `jsonb` and rejects malformed input at statement boundary. |
| T-02-01-02 | Trigger body calls `SET` / `SET LOCAL` ŌåÆ PgBouncer / pooling hazards (gate-01-01) | high | Code review + grep gate: `TriggerSQL.install_function/0` output must **not** contain `SET LOCAL` or `set_config` ŌĆö only `current_setting('threadline.actor_ref', true)`. |
| T-02-01-03 | `ON CONFLICT DO NOTHING` hides wrong `actor_ref` if GUC set late in txn | low | Document contract: host sets GUC before first audited write in the transaction; add test that early GUC wins. |

## Goal

Deliver `Threadline.Semantics.ActorRef` (struct + `Ecto.ParameterizedType`), Phase 2 DDL via a **new** idempotent migration file, extend `threadline_capture_changes()` so new `audit_transactions` rows receive `actor_ref` from the transaction-local GUC `threadline.actor_ref` (D-09 / CTX-03) while staying PgBouncer-safe (D-10: no `SET` in trigger), extend `Threadline.Capture.AuditTransaction` schema for new columns, update `Mix.Tasks.Threadline.Install` to emit the semantics migration when absent, and extend `Threadline.DataCase` cleanup order for `audit_actions`. Remove duplicate top-level scaffold modules **only if they exist** (guardrail).

End state: `mix compile --warnings-as-errors` and `mix verify.test` pass.
