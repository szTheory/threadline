## Goal

Add `Threadline.Semantics.AuditContext` struct (CTX-02). Implement `Threadline.Plug` that assigns `conn.assigns[:audit_context]` (CTX-01) using configurable `actor_fn: (Plug.Conn.t() -> ActorRef | nil)` and header/request-id extraction compatible with `Plug.RequestId` when present. Document and test that **PostgreSQL** propagation for actor uses the transaction-local GUC pattern from D-09 (call site documentation + link to `trigger_context_test`) — the Plug does not set PostgreSQL session state itself. Implement `Threadline.Job` pure helpers (`actor_ref_from_args/1`, `context_opts/2`) over plain maps — no Oban compile dep, no ETS, no process dictionary (CTX-05).
