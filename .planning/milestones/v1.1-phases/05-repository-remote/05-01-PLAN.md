## Tasks

### Task 1: REPO-02 ‚Äî `mix.exs` source URLs

**What to do**

1. Open `mix.exs` and confirm `@source_url` and `docs/0` ‚Üí `source_url:` use the canonical HTTPS URL without `.git` suffix: `https://github.com/szTheory/threadline`.
2. Confirm `package/0` ‚Üí `links: %{"GitHub" => @source_url}` (or identical string to `@source_url`).

**Read first**

- `mix.exs`
- `.planning/phases/05-repository-remote/05-CONTEXT.md` (D-01, D-02, D-05)

**Acceptance criteria**

- `grep -nF '@source_url "https://github.com/szTheory/threadline"' mix.exs` exits 0.
- `grep -nF 'source_url: @source_url' mix.exs` exits 0 **or** `grep -nF 'source_url: "https://github.com/szTheory/threadline"' mix.exs` exits 0.
- `grep -nF '"GitHub" => @source_url' mix.exs` exits 0 **or** `grep -nF '"GitHub" => "https://github.com/szTheory/threadline"' mix.exs` exits 0.

**Done condition:** All three greps succeed.

### Task 2: REPO-03 ‚Äî CI monitors `main`

**What to do**

1. Read `.github/workflows/ci.yml` header `on:` section.
2. Confirm both `push` and `pull_request` list `branches: [main]` only (same branch set as today).

**Read first**

- `.github/workflows/ci.yml`
- `.planning/phases/05-repository-remote/05-CONTEXT.md` (D-08)

**Acceptance criteria**

- `grep -A2 '^  push:' .github/workflows/ci.yml | grep -F 'branches: [main]'` exits 0.
- `grep -A2 '^  pull_request:' .github/workflows/ci.yml | grep -F 'branches: [main]'` exits 0.

**Done condition:** Both greps succeed.

---

### Task 3: REPO-01 ‚Äî `git remote` points at canonical repository

**What to do**

1. From repository root, run `git remote -v`.
2. Confirm `origin` fetch URL contains `szTheory/threadline` and uses either `https://github.com/` or `git@github.com:` form (D-05).
3. If `origin` is missing (bare import / tarball), **stop** and document in SUMMARY: "REPO-01 not applicable ‚Äî not a git clone"; that is a **blocker** for claiming Phase 5 complete ‚Äî maintainer must run from a clone with `origin` set.

**Read first**

- `.planning/phases/05-repository-remote/05-CONTEXT.md` (D-03‚ÄìD-05)

**Acceptance criteria**

- `git rev-parse --is-inside-work-tree` outputs `true`.
- `git remote | grep -qx 'origin'` exits 0.
- `git remote get-url origin | grep -E 'github\.com[:/]szTheory/threadline(\.git)?$|git@github\.com:szTheory/threadline\.git$'` exits 0.

**Done condition:** All three checks succeed.
