# Examples

The canonical Phoenix reference integration lives at **`examples/threadline_phoenix/`**. Start with [`threadline_phoenix/README.md`](threadline_phoenix/README.md) for the runnable example contract: prerequisites, `mix phx.new` regeneration flags, `mix threadline.install`, `mix threadline.gen.triggers`, and how to run **`mix phx.server`**, **`mix ecto.migrate`**, and **`mix test`** against **`threadline_phoenix_test`**.
